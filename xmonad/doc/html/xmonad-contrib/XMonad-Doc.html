Description

This is the main documentation module for the xmonad-contrib library. It provides a brief overview of xmonad and a link to documentation for configuring and extending xmonad.

A link to documentation describing xmonad internals is also provided. This module is mainly intended for those wanting to contribute code, or for those who are curious to know what's going on behind the scenes.

Synopsis
Overview

xmonad is a tiling window manager for X. The xmonad-contrib library collects third party tiling algorithms, hooks, configurations, scripts, and other extensions to xmonad. The source for this library is available from http://code.haskell.org/XMonadContrib via darcs: 

 darcs get http://code.haskell.org/XMonadContrib

Each stable release of xmonad is accompanied by a stable release of the contrib library, which you should use if (and only if) you're using a stable release of xmonad. You can find the most recent (Mar. 2008) tarball here: http://hackage.haskell.org/cgi-bin/hackage-scripts/package/xmonad-contrib-0.7
