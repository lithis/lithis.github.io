Robot Battle is a quickly-evolving programming game. The game allows players to design and build intelligent robots using a simple scripting language. Robot Battle is both an exciting game and an excellent way to learn programming. Most modern computer systems are based on event driven techniques very similar to those you will learn playing Robot Battle. Several schools are currently using the game as part of their introductory programming classes. Robot Battle was written by Brad Schick. The latest version is Robot Battle 1.31.
