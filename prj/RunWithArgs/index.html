run with arguments

version 0.1

run with arguments adds a context menu item to .exe files that allows you to pass command line arguments to them. it pops up a text input dialog and then runs the selected application with those command line arguments. it's quicker than opening a command prompt or using the run command on the start menu.

this is currently very diy. put the script in C:\Program Files\RunWithArgs\ and run the install registry file. you can run it from a different folder by editing the path in the registry file before running it. the shortcut key (currently g) can also be changed or removed by editing the registry file. to uninstall run with args, run the uninstall registry file.

entering no command line parameters and clicking ok is equivalent to clicking cancel. if you really want to run the program with no command line parameters, type a space in the input box.
